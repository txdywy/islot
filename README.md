# iSlot

一个纯前端 Web 版本老虎机游戏。定位是娱乐爽玩：用户可以随意给自己加金币，不涉及充值、提现或真实货币。

## 玩法

- 5 轴 3 行老虎机，5 条赔付线。
- 支持每线下注、自动旋转、极速模式、免费旋转。
- 3 个以上 Scatter 触发免费旋转。
- 多主题关卡：霓虹城、黄金埃及、玛雅神庙、极光冰原、龙焰工坊。
- 大奖时触发金币雨、火光、震屏、雪暴和主题粒子特效。

## 本地预览

这是静态站点，不需要构建步骤：

```bash
python3 -m http.server 5173
```

然后打开 `http://127.0.0.1:5173/`。

## GitHub Pages

仓库包含 `.github/workflows/pages.yml`。推送到 `main` 后，GitHub Actions 会把仓库根目录作为静态站点部署到 GitHub Pages。
